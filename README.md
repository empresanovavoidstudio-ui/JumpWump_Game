# JumpWump_Game
Jump Wump é um platformer 2D mobile colorido e expressivo, onde o jogador controla Pogo, um artista de circo que usa pulos, quicadas e impactos chamados WUMP para atravessar fases curtas e caóticas. Cada item desbloqueado libera uma mecânica única, criando novas formas de jogar e novos desafios.
